# 🔢 Decimal to Binary Converter (C Language)

## 📌 Description
This program converts a decimal (base 10) number into binary (base 2) using loops in C language.

## ⚙️ Logic Used

We divide the number by 2 repeatedly and store remainders:

- n % 2 → gives remainder (0 or 1)
- n / 2 → reduces number

Then we arrange remainders in reverse order.
